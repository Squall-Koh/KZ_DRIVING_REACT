③ TYPE 3 앱 생성 데이터 테이블

📱 앱이 직접 생성·기록하는 핵심 운행 데이터 테이블입니다.
BLE OBD 장치 감지 → 운행 시작/종료 → GPS 트랙 기록의 흐름으로 데이터가 쌓이며, 출퇴근 기록(work_days), 개별 운행(trips), GPS 궤적(track_data), 알림(notifications)으로 구성됩니다.

④ TYPE 4 로그 테이블

🔍 앱 동작 분석 및 디버깅 목적의 로그 테이블입니다.
BLE 스캔 이벤트, 장치 감지 여부, RSSI 신호 강도, 배터리 수준 등을 주기적으로 기록하여 OBD 연결 품질 및 운행 상태 전환 이력을 추적합니다. 현재 34건 기록됨.
